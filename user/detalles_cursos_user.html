Descripción

La programación web es una herramienta muy importante hoy en día, ya que permite generar un dialogo constante y amigable entre los usuarios y la información que está en el sitio web. Por ello, el curso tiene como objetivo programar a través de los lenguajes HTML5, CSS3, Java Script, PHP y MySQl, para la creación de sitios web.
